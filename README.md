# Hastane Acil Takip Sistemi

## Proje Hakkında
Bu proje hastane acil servisinde hasta takip işlemlerini yönetmek amacıyla geliştirilmiştir.

## Özellikler
- Hasta kayıt işlemleri
- Acil durum takibi
- Hasta bilgilerini listeleme
- Güncelleme işlemleri

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Projeyi Çalıştırma
1. Dosyaları indiriniz
2. index.html dosyasını açınız
3. Sistem çalışacaktır
