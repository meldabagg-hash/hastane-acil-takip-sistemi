# Akıllı Acil Ön Değerlendirme ve Sıra Takip Sistemi

## Proje Hakkında
Bu proje, hastane acillerindeki yoğunluğu azaltmak amacıyla geliştirilmiştir. Grip, öksürük ve boğaz ağrısı gibi acil olmayan durumlarda hastaların hastanede uzun süre beklemesini önlemek için online ön değerlendirme ve sıra sistemi sunmaktadır.

## Projenin Amacı
- Acil yoğunluğunu azaltmak
- Hastaların gereksiz bekleme süresini önlemek
- Bulaşıcı hastalık riskini azaltmak
- Hastaların telefondan ön değerlendirme almasını sağlamak

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Özellikler
- Online sıra alma
- Şikayet seçme sistemi
- Aciliyet değerlendirmesi
- Ortalama bekleme süresi gösterme
- Basit kullanıcı arayüzü
